Usage:

he2shadercompiler <command> <file> <options>

commands:
    compile             Compiles an HLSL file
    info                Prints info about a compiled HE2 shader file
    dump                Dumps the contents of a compiled HE2 shader into several files
    extract             Dumps shader object files embedded within an executable into files
    help                Shows this

options:
    -h --help            Print help and options specific to a command
    -o <filename>        Output filename
    -od <directory>      Output directory (automated filenames)
